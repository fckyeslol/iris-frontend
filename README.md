Iris Classifier Frontend

This is the frontend of my Iris Classifier project, built with HTML, CSS, and JavaScript.

Overview

The interface allows users to input plant features:

Sepal length
Sepal width
Petal length
Petal width

It sends these values to the backend API and displays:

The predicted iris species
The probability of each possible class
Tech Stack
HTML
CSS
JavaScript
How it Works
User inputs the plant features
The app sends a request to the backend API
The response is processed and displayed dynamically on the page
API Connection

The frontend connects to a FastAPI backend hosted on Railway.

Deployment

The frontend is deployed on Vercel.

Notes
This repository contains only the frontend
The backend is hosted separately and handles all model inference
